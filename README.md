# Theme-Manager
# A Theme Manager To Manage Gtk Themes As well as icons/gnome-shell/cursor themes
Currently supports 
- Gtk Themes
- Icon Themes
- Shell Themes (For gnome only after installing user-themes extension)
- Cursor Themes
